
#  trim.it

trim.it is a web app that allows users to shorten long links into more concise and shareable ones, complete with custom slug and QR Code generator features.

trim.it adalah sebuah web app yang memungkinkan pengguna untuk mempersingkat link panjang menjadi link yang lebih ringkas dan dapat dibagikan, lengkap dengan fitur custom slug dan QR Code generator.

---

## Features

-  URL Shortener
-  Custom Slug (ex: trim.it/portofolio)
-  QR Code Generator
-  Recent Links Section

---

##  Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (State Management)
